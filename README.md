# Blinkit-Sales-Data -Analysis (Interactive Dashboard creation using MS Excel)
## Project Objective
The Blinkit Store wants to create a comprehensive analysis of Blinkit's sales report from 2011 to 2022. So that, the owner of the Blinkit store can understand their customers and grow more sales in 2023 & 2024.
## Dataset used
<a href= "https://github.com/AmishaMishra123/blinkit-sales-analysis-excel-dashboard/blob/main/New-Blinkit.xlsx">Blinkit DataSet</a>

## Questions (KPIs)
-	What is the total revenue/sales shown on the dashboard?
-	What is the average revenue per sales?
-	Count of different items sold.
-	The average customer rating for the items sold.
-	Which fat content category (Low Fat, Regular, etc.) generates the highest total sales?
-	Which item type contributes the most to total sales?
-	Compare total sales across different outlets segmented by fat content.
-	How does total sales vary across different outlet establishment year categories?
-	Percentage of Sales by Outlet Size
-	How do total sales differ across various outlet locations?
-	How do key performance metrics vary across different outlet types?

## Process
-	Verify data for any missing values and null, and sort out the same.
-	Made sure data is consistent and clean with respect to data type, data format and values used.
-	Created pivot tables according to the questions asked.
-	It operated by using filters so that the data can be sliced and can see the insights at different angles or different filter levels.
-	Merge all pivot tables into one dashboard and apply slicer to make dynamic dashboard.

## Project Insights
-	Total sales are strong at $1.20M with good overall performance. 
-	Average Sales of $141 showing how much revenue each item generates on average.
-	Customers prefer regular-fat items and essential categories like Fruits & Vegetables. 
-	Medium-sized outlets and Tier 3 locations generate the highest sales.
-	Seafood and other low-demand categories show weaker performance. 
-	Sales peaked in 2018 and remained stable afterward.

## Final Conclusion
The Blinkit Sales Analysis shows strong overall performance, led by regular-fat products, medium-sized outlets, and Tier 3 locations. Key categories like Fruits & Vegetables and Snack Foods contribute the most to revenue, reflecting customer preference for essential and frequently purchased items. Lower-performing categories such as Seafood highlight opportunities for better promotion, inventory adjustments, or pricing improvements. Sales peaked in 2018 and have remained stable in recent years. Overall, the dashboard provides valuable insights into customer behaviour and outlet efficiency, helping Blinkit focus on high-performing areas while improving weaker segments to support continued business growth.
